Integrate the MagnaPass booking widget on your WordPress website.

== Description ==

The MagnaPass widget allows you to embed our booking widget directly on to your wordPress website.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

Now on any page you want to display the widget, use the following "shortcode":

[magnapass partner="&lt;replace me&gt;"]

Replace the <replace me> with your partner key, available from your MagnaPass dashboard.
